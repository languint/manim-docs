Manim animations are contained in `Scenes`. Each scene must be decorated with `@Manim.Decorators.Scene`.

```sh frame=none
import { Manim } from "@rbxts/manim";
import { Workspace } frorm "@rbxts/services";

@Manim.Decorators.Scene(
	{
		name: "VectorTestScene",
		destroyOnCompleted: false,
	},
	Workspace,
)
export class MyScene extends Manim.SceneWithCamera {
    construct(): void {
	}

	tick(dt: number): void {
		this.tickAllChildren(dt);
	}
}
```

Notice that we have to define the `tick()` method. This is optional, and you can choose whether or not to have a static scene.
